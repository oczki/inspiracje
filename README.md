## Summary

Magazyn Inspiracji (Polish for "inspiration depot") is a web application that outputs Polish words, split into categories.

It's intended for improv groups that need some anchor words for practice. They would usually get such data from the audience during performances, but there's no audience during private training sessions â€“ Magazyn Inspiracji fills that hole with over 1500 ideas.
